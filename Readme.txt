Instructions to Run a JAR File on PC

Install Java Development Kit (JDK):

If you don't have the JDK installed, download and install it from Oracle's official site or OpenJDK.
Verify Java Installation:

Open Command Prompt (Windows) or Terminal (Mac/Linux).
Type java -version and press Enter. You should see the installed version of Java.
Download the JAR File:

Download the JAR file from the GitHub repository.
Navigate to the Directory:

Open Command Prompt (Windows) or Terminal (Mac/Linux).
Navigate to the directory where the JAR file is located using the cd command. For example:

cd path/to/your/jarfile
Run the JAR File:

Use the following command to run the JAR file:

java -jar yourfile.jar
Replace yourfile.jar with the name of your JAR file.
